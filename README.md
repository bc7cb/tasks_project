# تطبيق قائمة المهام (Todo App)

تطبيق ويب تفاعلي لإدارة المهام اليومية باستخدام HTML وCSS وJavaScript.

## المميزات

- إضافة مهام جديدة
- تحديد حالة المهمة (مكتملة / قيد الانتظار)
- تعديل المهام
- حذف المهام
- تصفية المهام (الكل / قيد الانتظار / مكتملة)
- إحصائيات عن عدد المهام
- حفظ المهام في التخزين المحلي للمتصفح
- تصميم متجاوب يعمل على جميع الأجهزة
- واجهة مستخدم عربية

## كيفية الاستخدام

1. **إضافة مهمة جديدة**: اكتب نص المهمة في حقل الإدخال واضغط على زر "إضافة" أو اضغط على مفتاح Enter.
2. **تحديد حالة المهمة**: انقر على خانة الاختيار بجانب المهمة لتحديدها كمكتملة أو قيد الانتظار.
3. **تعديل المهمة**: انقر على زر التعديل (أيقونة القلم) لتعديل نص المهمة.
4. **حذف المهمة**: انقر على زر الحذف (أيقونة السلة) لحذف مهمة معينة.
5. **تصفية المهام**: استخدم أزرار التصفية في الأعلى لعرض جميع المهام أو المهام قيد الانتظار أو المكتملة فقط.
6. **حذف المهام المكتملة**: انقر على زر "حذف المكتمل" لحذف جميع المهام المكتملة.
7. **حذف جميع المهام**: انقر على زر "حذف الكل" لإزالة جميع المهام.

## التقنيات المستخدمة

- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (ES6+)

## التخصيص

يمكنك تخصيص التطبيق بسهولة عن طريق:

- تغيير الألوان في ملف `style.css`
- إضافة ميزات جديدة في ملف `script.js`
- تعديل النصوص والعناصر في ملف `index.html`
